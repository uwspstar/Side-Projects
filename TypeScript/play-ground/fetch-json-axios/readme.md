### project init

- \$ mkdir fetch-json-axios
- \$ cd fetch-json-axios
- \$ npm init -y
- \$ npm install axios
- \$ touch .gitignore
- \$ touch index.ts

---

### run tsc index.ts will generate index.js (same as ts-node)

- \$ tsc index.ts // will generate index.js
- \$ node index.js
- \$ ts-node index.ts// combine 2 command in one
