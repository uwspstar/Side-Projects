- https://www.youtube.com/watch?v=5YoKjdUf3-8&t=321s

---

- \$ pm2 start ...
- \$ pm2 list
- \$ pm2 restart <AppName>
- \$ pm2 stop <AppName>
- \$ pm2 info <AppName>
- \$ pm2 log <AppName>
- \$ pm2 monitor <AppName>
