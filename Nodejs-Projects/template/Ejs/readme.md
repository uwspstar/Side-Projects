### folder structure
```
$ touch readme.md
$ touch .env
$ touch .gitignore
$ touch server.js
$ mkdir views 
$ touch views/index.ejs
$ mkdir views/template
$ touch views/template/head.ejs
$ touch views/template/foot.ejs
$ touch views/template/nav.ejs
```

```
$ npm install ejs
```
